##  Testes Automatizados (Parte 4)
Foram desenvolvidos dois testes utilizando **Selenium WebDriver (Python)**:
1. Teste de validação da página de Login  
2. Teste de registro de denúncia  

As páginas HTML incluídas simulam telas reais do sistema, permitindo testes locais.

##  Como executar os testes Selenium
### 1. Instalar dependências:
```
pip install selenium
```

### 2. Baixar ChromeDriver:
https://googlechromelabs.github.io/chrome-for-testing/

### 3. Executar testes:
```
python teste1_login.py
python teste2_denuncia.py
```
